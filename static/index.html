Welcome to Wasm-Mazes!

Here you can generate mazes directly to pdf files and print them.
You can use one of the predefined difficulty settings or customise your own.

This maze generator uses WebAssembly written in Rust to generate mazes fully in your browser,
no server needed. If you are interested how this works, then check out the repository here Kazhuu/wasm-mazes.

Generating very big mazes might be slow, so be patient.

Have fun!
